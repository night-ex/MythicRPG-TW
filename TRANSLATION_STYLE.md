# Translation Style

## 翻譯規範

- 保留 `%s`、`%d`、`{}` 等佔位符，不翻譯、不刪除，且數量與形式必須和 `en_us.json` 一致。
- 不翻譯 Mod ID。
- 不翻譯 NBT。
- 不翻譯 Resource Location。
- 保留並統一使用 Minecraft 官方術語。
- UI 採用自然、清楚的繁體中文。
- 成就名稱可以意譯，但必須保留原意與相關典故。
- Lore 以模組世界觀與敘事語氣優先。
- Boss 名稱必須依照 `TRANSLATION_GLOSSARY.md` 保持一致。

## 詞彙流程

1. 翻譯前先查閱 `TRANSLATION_GLOSSARY.md`。
2. 已有詞條必須沿用，不得自行改譯。
3. Glossary 沒有的新專有名詞，必須先確定統一譯名並新增詞條。
4. 新詞條必須包含英文原文、中文譯名、首次使用模組、是否為 Minecraft 官方譯名及必要備註。
